# Dropdown Menu, Modal Popup & Progress Bar

A simple frontend project built using HTML, CSS, and JavaScript that demonstrates three important UI components:

- Dropdown Menu
- Modal Popup
- Progress Bar

## Features

### Dropdown Menu
- Toggle menu on button click
- Show/Hide functionality

### Modal Popup
- Open modal using button
- Close modal using close button
- Overlay background

### Progress Bar
- Increase progress dynamically
- Smooth animation
- Percentage display

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Folder Structure

project/

├── index.html

## How to Run

1. Download the project.
2. Open in VS Code.
3. Install Live Server.
4. Run index.html using Live Server.
